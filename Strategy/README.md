# Strategy

Business strategy documents for Long Tales. Anything that shapes how the business is built rather than what it sells.

Likely subfolders, created as work demands them:

- `ops-strategy/` — sourcing, batch operations, rider logistics, society rollout sequencing.
- `financial-projections/` — monthly P&L models, scenario planning, working capital needs.
- `positioning/` — competitive landscape work, white-space mapping, narrative experiments.
- `launch-plan/` — pilot design, society-by-society rollout, sampling math, conversion targets.
- `pricing/` — SKU pricing thinking, discount discipline, gifting and bulk SKU economics.

Drafts in progress live in the scratch working directory. Only versions Saurabh has reviewed move here.
