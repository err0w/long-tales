# Collaterals

Approved brand assets only. Nothing goes in this folder until Saurabh has signed it off.

What belongs here: packaging labels, jar stickers, pamphlets, landing pages, photos, logos, batch card templates, WhatsApp order scripts, pricing cards.

What does not belong here: drafts, explorations, options being compared, moodboards, anything still under review. Those live in the scratch working directory until approved.

Naming convention: `YYYY-MM-DD_asset-name_vN.ext`. Version numbers go up with each approved revision, they do not reset.
